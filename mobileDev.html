Leafio

Leafio is a mobile application designed for the detection of plant leaf diseases. Users can easily upload images of plant leaves, which are then seamlessly transmitted to our server. The server is equipped with a powerful Convolutional Neural Network (CNN) model designed for plant leaf disease detection. The results are then retrieved by the app and presented to users in a user-friendly display.

Converse Messenger

Converse Messenger is a dynamic real-time chat application that integrates the Firebase database to provide a robust platform for user authentication, private one-to-one messaging, and secure user data storage; and uses Firebase Firestore for storage of users' profile photos. Additionally, it employs RecyclerViews to elegantly organize and display chat conversations and contact information, offering users a cohesive and meaningful interface.

  • GitHub

NoteIt

NoteIt is a versatile notes application designed for note creation, search functionality, and storage. Making use of Room Database, it ensures that the notes are safely stored on the device. Through the implementation of recycler views, NoteIt presents the saved notes in a visually captivating manner
