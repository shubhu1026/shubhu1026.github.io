Gravebound Adventures

Gravebound Adventures, a Unity game from a three-day jam around 'Two Dimensions,' challenges players in a unique 2D puzzle platformer where parkour meets critical thinking as they guide the character to their grave by mastering dimension-switching.

To play the game, click the "itch.io" link, or explore the code on GitHub. Your choice!

Powers of the Wild

Powers of the Wild is an engaging 3D puzzle game where each respawn grants players a random set of animal abilities, including Grasshopper's super jump, Mole's burrowing, Snail's adhesive prowess, Mouse's shrinking, and more. Strategically harness these talents to conquer challenges and achieve your objectives.

Experience the game on "itch.io" or inspect the code on GitHub. It's up to you!

  • itch.io
  • GitHub

Hungrylodon

Hungrylodon is a Unity-powered classic grow-and-eat game infused with unique abilities. Crafted during a game jam, players take on the role of a shark, consuming smaller fish, gathering valuable power-ups, and dodging obstacles to achieve the highest score.

Try the game on "itch.io" or check out the code on GitHub. Your pick!
