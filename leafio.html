Leafio

Leafio is an Android app designed primarily for farmers to help them reduce their workload.
Farmers can use the app to take a photo of their plant's leaves, and the app will tell them whether the plant is healthy or diseased.

The Main Activity offers the user the option of accessing an image from their gallery or capturing an image with their camera. The user can then click the Predict button to make an API call to our Flask App, which is hosted on Heroku. Our Convolutional Neural Network (CNN) Model processes the image after it has been uploaded to the server. After the Model has finished processing and identifying the leaf disease, the result is received in JSON format by the Android app.

Retrofit was used to make a server call and receive a response via an API (Application Program Interface) call. Following that, GSON was used to parse the response.
